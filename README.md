## Image Lazy Loading

### Introduction
Implements images lazy loading for any image on the forum.

I'm not sure it doesn't break anything.

## 3rd party
Uses the [Lazy Load Plugin for jQuery](https://www.appelsiini.net/projects/lazyload)
